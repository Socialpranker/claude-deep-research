# Open data aggregators

Большие порталы открытых данных. Использовать когда ищешь raw datasets для анализа или когда первичный источник неизвестен.

### Data.gov (US)
**URL:** data.gov
**Access:** OPEN
**What:** 300k+ US federal datasets aggregated
**When:** searching for US government datasets когда специфическое агентство неизвестно
**How:** Search by topic; download CSV/XML/API
**Quality:** A (gov source); freshness varies by dataset

### EU Open Data Portal
**URL:** data.europa.eu
**Access:** OPEN
**What:** 1M+ EU datasets from EU institutions and member states
**When:** EU-related research, cross-country comparisons
**How:** Search and filter by country/topic; downloads available

### Kaggle Datasets
**URL:** kaggle.com/datasets
**Access:** OPEN
**What:** Community-contributed datasets (mostly ML-focused)
**When:** when you need pre-cleaned data для analysis, ML examples
**Quality:** B-C (community-uploaded; check author reputation)

### Google Dataset Search
**URL:** datasetsearch.research.google.com
**Access:** OPEN
**What:** Meta-search across thousands of dataset repositories
**When:** finding datasets когда не знаешь where they're hosted
**How:** Search like Google Scholar

### HuggingFace Datasets
**URL:** huggingface.co/datasets
**Access:** OPEN
**What:** ML-focused datasets, especially NLP/vision
**When:** ML research, especially LLM-relevant data

### UN Data
**URL:** data.un.org
**Access:** OPEN
**What:** UN system data — population, health, education
**When:** global UN-tracked indicators

### OpenStreetMap
**URL:** openstreetmap.org
**Access:** OPEN
**What:** Open geographic data — maps, POIs
**When:** geographic analysis, location data

### Open Data Network
**URL:** opendatanetwork.com
**Access:** OPEN
**What:** Meta-aggregator across multiple open data portals
**When:** searching across portals

### ProPublica Data Store
**URL:** propublica.org/datastore
**Access:** OPEN
**What:** Journalism-grade cleaned datasets (US-focused)
**When:** investigative journalism research

### CKAN Portals
**URL:** various — `ckan.org/instances`
**Access:** OPEN
**What:** CKAN is open-source data portal software — hundreds of instances
**When:** finding specific country/city data portals

### ICPSR (academic data)
**URL:** icpsr.umich.edu
**Access:** PARTIAL (registration for some)
**What:** Academic research datasets (social sciences)
**When:** academic research with real-world data

### Awesome Public Datasets
**URL:** github.com/awesomedata/awesome-public-datasets
**Access:** OPEN
**What:** Curated GitHub list of public datasets by topic
**When:** discovering data sources you didn't know exist

## Approach

Используй aggregators когда:
1. Ты знаешь нужен dataset но не знаешь где
2. Хочешь сравнить альтернативные источники
3. Нужна dataset для own analysis (не just facts)

НЕ используй когда:
- Знаешь конкретный first-party источник (иди прямо к нему)
- Нужны parsed numbers (aggregators дают raw, не narratives)
