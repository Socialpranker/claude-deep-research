# Transport & travel

Aviation, shipping, road, rail, tourism.

## Aviation

### FlightRadar24
**URL:** flightradar24.com
**Access:** OPEN (free tier)
**What:** Live + historical flight tracking
**When:** specific flight analysis, airline operations

### OAG / Cirium
**URL:** oag.com, cirium.com
**Access:** PAYWALL with press releases
**What:** Airline schedules, capacity, on-time performance
**When:** airline operations analysis (через press releases)

### IATA Reports
**URL:** iata.org/en/iata-repository/publications
**Access:** OPEN partial
**What:** Aviation industry reports, traffic, statistics
**When:** global aviation industry overview

### ICAO Data
**URL:** icao.int/safety/iStars
**Access:** OPEN
**What:** Global civil aviation safety + traffic data
**When:** safety claims, global traffic

### FAA (US)
**URL:** faa.gov/data_research
**Access:** OPEN
**What:** US aviation data
**When:** US-specific aviation

### Eurocontrol
**URL:** eurocontrol.int/publications
**Access:** OPEN
**What:** EU air traffic data
**When:** EU aviation analysis

## Combining patterns

**Aviation industry analysis:** IATA reports + ICAO data + FlightRadar24 (specific routes) + national CAA data (e.g., FAA, EASA) + Cirium press releases для recent

**Container shipping market:** Drewry indices (рост/падение) + CTS volumes + WTO trade context + Drewry weekly reports

**Tourism destination:** UNWTO (country) + WTTC (economic impact) + local tourism board + Airbnb data via AirDNA + flight data via flight aggregators
