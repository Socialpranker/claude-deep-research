# Cybersecurity

## Tier 1

### Verizon DBIR (Data Breach Investigations Report)
**URL:** verizon.com/business/resources/reports/dbir
**Access:** OPEN
**What:** Annual analysis of thousands of data breaches — attack patterns, actors, industries
**When:** breach pattern analysis, ransomware/attack trend claims
**Quality:** A — most-cited security report

### IBM Cost of a Data Breach Report
**URL:** ibm.com/reports/data-breach
**Access:** OPEN
**What:** Annual report on breach costs by industry, country, type
**When:** breach cost claims, security ROI argument

### Cybersecurity Ventures
**URL:** cybersecurityventures.com
**Access:** OPEN
**What:** Cybersecurity market predictions, job projections
**When:** market sizing (with caveat — bullish projections)
**Quality:** B (bullish on cybersecurity growth)

## Tier 2

**URL:** enisa.europa.eu
**Access:** OPEN

**When:** EU cybersecurity context

### CISA (US Cybersecurity & Infrastructure Security Agency)
**URL:** cisa.gov
**Access:** OPEN

## Combining patterns

**Threat landscape analysis:**
Verizon DBIR + ENISA + Mandiant M-Trends + CrowdStrike Global Threat + national CERTs

**Breach cost analysis:**
IBM Cost of Data Breach + Ponemon Institute (related research) + insurance industry data + DBIR breach analysis

**Specific vulnerability:**
CVE/NVD + CISA KEV catalog + vendor advisory + exploit DB (`exploit-db.com`)

**Cybersecurity market sizing:**
Cybersecurity Ventures (bullish) + Gartner Security (per press) + IDC + analyst consensus
