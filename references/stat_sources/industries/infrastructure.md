# Public Infrastructure

## Tier 1

### OECD Infrastructure
**URL:** oecd.org/infrastructure-finance
**Access:** OPEN
**What:** Global infrastructure investment data, project trends
**When:** infrastructure investment context

### G20 Global Infrastructure Hub
**URL:** gihub.org/resources/data
**Access:** OPEN
**What:** Infrastructure needs, investment forecasts, project database
**When:** global infrastructure gap analysis

### World Bank Infrastructure
**URL:** worldbank.org/en/topic/infrastructure
**Access:** OPEN
**What:** Infrastructure financing data + research
**When:** developing country infrastructure

## Tier 2

### CG/LA Infrastructure (Strategic 100)
**URL:** cglainfrastructure.com
**Access:** Reports partial
**What:** Annual ranking strategic infrastructure projects
**When:** specific projects research

### ASCE Report Card (US)
**URL:** infrastructurereportcard.org
**Access:** OPEN
**What:** American Society of Civil Engineers — US infrastructure grades by category
**When:** US infrastructure quality assessment

### McKinsey Global Institute Infrastructure
**URL:** mckinsey.com/mgi
**Access:** OPEN
**What:** Infrastructure productivity, investment research
**When:** macro infrastructure analysis

### IJ Global
**URL:** ijglobal.com — partial
**Access:** PARTIAL
**What:** Infrastructure project finance database
**When:** project finance research

### PPP (Public-Private Partnership) data

- **EPEC (EIB EPEC)** — eib.org/en/epec — EU PPP data
