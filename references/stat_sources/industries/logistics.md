## Specific

### Shipping rates
- **Baltic Dry Index** (Baltic Exchange) — bulk shipping
- **Harpex (Container Shipping)** — partial
- **SCFI (Shanghai Containerized Freight Index)** — OPEN

### Port stats
- **Lloyd's List 100 Top Ports** — annual ranking — OPEN
- **AAPA (American Association of Port Authorities)** — US port data
- **ESPO** (European Sea Ports) — EU port data

### Last-mile / parcel
- **Pitney Bowes Parcel Shipping Index** — annual OPEN
- **ShipMatrix** — US parcel benchmarks — partial

## Combining patterns

**Container shipping market:**
Drewry WCI + Container Trade Statistics + SCFI/HARPEX + Lloyd's List port rankings + WTO trade context

**US logistics cycle:**
Cass Freight Index + FreightWaves + ATA Truck Tonnage Index + BTS

**Country logistics quality:**
World Bank LPI + national port stats + trade data context (WTO/Comtrade)

**Air cargo:**
IATA Cargo + flight data (FlightRadar24) + airport-specific data
